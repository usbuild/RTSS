Rail Ticket Service System

Dependency
==============================================
gtk3 & gtk3-dev
packages in Ubuntu is `libgtk-3-0` and `libgtk-3-dev`

Build
==============================================
* `make` to build whole system
* `make client` to build client
* `make server` to build server
